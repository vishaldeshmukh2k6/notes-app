# 📝 Flask Notes App

A simple web-based Notes application built using **Flask**, **SQLite**, and **Bootstrap**.  
You can add and delete short notes easily!

---

## 📌 Features

- 🧾 Add a new note
- 📜 View all notes
- ❌ Delete a note
- 🎨 Clean and responsive UI using Bootstrap

---

## 🔧 Tech Stack

- Python 🐍
- Flask 🌐
- SQLAlchemy 🗃️
- SQLite 💾
- HTML + Bootstrap 🎨
